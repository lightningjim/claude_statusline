_BAR_WIDTH = 20   # bar is 20 cells wide; gradient must also total exactly 20 cells.
RESET = "\033[0m"
color_for(pct, warn, crit) -> str   # threshold color for the filled run.

Partial-block set (left-aligned, ascending 1/8 → 7/8), exactly 7 glyphs:
  ▏ (1/8) ▎ (2/8) ▍ (3/8) ▌ (4/8) ▋ (5/8) ▊ (6/8) ▉ (7/8)
Full cell: █ (8/8). Empty track for gradient: " " (single space).

Eighth math (Claude's Discretion in CONTEXT.md, suggested):
  total_eighths = round(pct/100 * _BAR_WIDTH * 8)   # clamp to [0, _BAR_WIDTH*8]
  full_cells    = total_eighths // 8
  remainder     = total_eighths % 8                 # 0..7
  boundary glyph = partial[remainder-1] when remainder>0 else none
  blank cells fill the remaining width so total cells == _BAR_WIDTH
  Edge: 0% → 0 full, 0 remainder → 20 blanks. 100% → 20 full, 0 remainder → 20 █ (no stray partial).

Config selection (for the human-verify task):
  TOML at ~/.claude/claude-statusline/claude-statusline.toml, key [display] bar_style = "shade" | "solid" | "solid-dim" | "gradient" (D-08).
  Default is "shade" (D-09). The script reads one JSON object from stdin and prints the status line(s).
