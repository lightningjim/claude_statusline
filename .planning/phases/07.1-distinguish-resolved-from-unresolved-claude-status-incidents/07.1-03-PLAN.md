| Boundary | Description |
|----------|-------------|
| cached feed labels → terminal (render) | RAW incident/resolved titles from cache are sanitized and printed by `_claude_status_segment` to the statusline |
| cached feed labels → terminal (CLI) | RAW titles printed by `_print_status_incidents` for `--status-incidents` |
| dismissal store (local JSON) → render/CLI | Read to re-apply suppression and compute STATE; corrupt store must degrade |
