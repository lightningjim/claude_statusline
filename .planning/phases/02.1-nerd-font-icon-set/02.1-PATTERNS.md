```python
def _icon_to_emoji(text_description: str, icon_url: str) -> str:
    desc = (text_description or "").lower()
    icon_path = (icon_url or "").lower()
    # is_night flag: NWS URLs encode e.g. /icons/land/night/skc
    is_night = "/night/" in icon_path

    for keywords, emoji in _NWS_ICON_MAP:
        for kw in keywords:
            if kw in desc or kw in icon_path:
                if is_night and emoji == "☀️":
                    return "🌙"
                return emoji
    return "🌡️"  # fallback: thermometer
```

**What changes for Phase 02.1:**
- Rename `_NWS_ICON_MAP` → `_NWS_ICON_MAP_EMOJI` (retained as-is for `icon_set="emoji"` fallback).
- Create `_NWS_ICON_MAP_NERD` using the same `list[tuple[tuple[str, ...], str]]` type, with expanded NWS token vocabulary (distinct glyphs for `tsra`, `rasn`, `fzra`, `sn`, `ra`, `fg`, `wind`, `skc/few/sct/bkn/ovc` day vs night) mapped to `wi-*` PUA codepoints.

- The `is_night` flag and the first-match `for/for` loop are UNCHANGED.
- For the nerd path, when `is_night` and the matched glyph is a clear/sun glyph, branch into moon-phase computation (see `_sun_segment` astral pattern below) instead of returning a fixed `🌙`.
- The fallback glyph (`🌡️`) changes to the nerd-path fallback thermometer codepoint; keep `🌡️` for emoji path.

**First-match ordering rule** (line 301–305 comment): The more-specific tokens (`partly cloudy`, `partly sunny`, `mostly sunny`, `mostly clear`, `few`, `scattered`) MUST appear before the broad `cloudy`/`sunny` entries. Preserve this ordering discipline in the nerd table.

```python
def _deep_merge(base: dict, override: dict) -> dict:
    result = copy.deepcopy(base)
    for key, val in override.items():
        if key in result and isinstance(result[key], dict) and isinstance(val, dict):
            result[key] = _deep_merge(result[key], val)
        else:
            result[key] = val
    return result
```

```python
def _wx_color(condition_type: str) -> str:
    """Return semantic ANSI color for a weather condition category (D-08)."""
    if condition_type == "storm":
        return MAGENTA
    if condition_type == "rain":
        return BLUE
    if condition_type == "snow":
        return CYAN
    if condition_type == "fog":
        return GRAY
```
