## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| status.claude.com network → refresh child | Untrusted JSON (free-text incident/maintenance titles, component names, arbitrary types/sizes) crosses into the process |
| refresh child → cache.json file | Derived status (incl. raw title) is persisted; render later reads it |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Tampering | incident/maintenance title text (rendered later) | mitigate | Store RAW title in cache here; ANSI/control-char sanitization is enforced on the render path (Plan 02, T-06-04). This plan's derivation MUST NOT itself emit the title to a terminal. |
