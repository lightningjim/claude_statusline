## Decisions Made

- Chose nf-fa-* range (U+F0xx) for all four alert glyphs — same proven-safe range as existing _NF_GSD_* constants, all present in the installed JetBrains Nerd Font cmap
- fa-warning (U+F071) for Warning: glyph name literally contains "warning" — perfect semantic match
- fa-eye (U+F06E) for Watch: "watching" / observing — correct operational meaning
- fa-info_circle (U+F05A) for Advisory: information/advisory circle — correct meaning
- fa-bell (U+F0F3) for Statement/Other: general notification — neutral, non-alarming
- `_alert_color({})` returns RESET (Statement/Other → RESET, not YELLOW). YELLOW is only the `except Exception` fallback, not the Statement/Other default. Tests adjusted to verify no-raise + non-empty string rather than asserting YELLOW equality for empty-dict input.
