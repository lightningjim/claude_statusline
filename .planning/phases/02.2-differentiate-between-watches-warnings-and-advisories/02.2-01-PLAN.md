<objective>
Build the pure, testable classification + color + selection layer for hazard-class-aware alerts, reshaping the three existing severity-only functions into class-first ones. This is the foundation Plan 02's render wiring consumes.

Purpose: Decouple the alert signal from noisy CAP severity — encode action level (Warning/Watch/Advisory/Statement) as hue, immediacy (urgency+certainty) as intensity, and rank primary selection class-first. The user is a degreed meteorologist; W/W/A are real operational distinctions, not severity synonyms.
Output: New constants + four helpers (`_classify_alert_class`, `_alert_intensity`, reshaped `_alert_color(alert)`, reshaped `select_alert`) in `claude-statusline.py`, fully unit-tested, with the new glyph constants validated against the installed font cmap.
</objective>
