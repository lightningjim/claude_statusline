| T-02-02 | Elevation/Tampering | install.py subprocess (venv build, pip install) | mitigate | Invoke subprocess.run with a FIXED argument-vector list and check=True; never shell=True, never string-interpolate a path into a shell command |

| T-02-04 | Tampering | default config copy in install.py | mitigate | Copy default config only when CONFIG_PATH does not already exist — never clobber a user's edited config |
| T-02-05 | Denial of Service | astral/requests import failure | mitigate | Guarded import sets _WEATHER_OK False; _weather_segment returns None so the Phase 1 bar renders untouched (D2-12) |
| T-02-SC | Tampering | pip install requests/astral | mitigate | Both packages verified legitimate below; no [ASSUMED]/[SUS]/[SLOP] — no blocking checkpoint required |
